Automatizaciones sencillas con Python

  • Alberto Iturralde
  • Python

La automatización de tareas sencillas se ha convertido en una necesidad en el mundo digital actual, donde el tiempo es un recurso valioso. Python, con su versatilidad y facilidad de uso, emerge como una herramienta poderosa para simplificar la automatización de tareas cotidianas. En este artículo, exploraremos cómo puedes aprovechar Python para crear scripts y programas que realicen acciones repetitivas de manera eficiente, ahorrando tiempo y minimizando errores.

¿Por qué Python?

Python se ha convertido en uno de los lenguajes de programación más populares gracias a su sintaxis clara y legible. Estas características hacen que sea accesible para principiantes y, al mismo tiempo, potente para los desarrolladores experimentados. Además, la amplia variedad de bibliotecas y módulos disponibles facilita la automatización de una amplia gama de tareas.

Automatización de Tareas Repetitivas con Scripts en Python

Uno de los usos más comunes de Python en la automatización es la creación de scripts para realizar tareas repetitivas. Imagina tener que renombrar un conjunto de archivos de manera sistemática o descargar archivos de una página web de forma regular. Con Python, puedes escribir un script simple para manejar estas tareas.

                  
  import os

  # Obtener la lista de archivos en el directorio actual
  archivos = os.listdir()
  
  # Renombrar archivos agregando un prefijo
  for archivo in archivos:
      nuevo_nombre = "prefijo_" + archivo
      os.rename(archivo, nuevo_nombre)
  
  print("Tareas de renombrado completadas con éxito.")

Este es solo un ejemplo muy básico. La versatilidad de Python te permite abordar una variedad de tareas con scripts personalizados.

Automatización de Procesos con Programas en Python

Además de los scripts, Python también puede utilizarse para desarrollar programas más complejos que automatizan procesos completos. Por ejemplo, si trabajas con hojas de cálculo regularmente y necesitas realizar manipulaciones específicas, un programa Python puede ser la solución.

                  
  import pandas as pd

  # Cargar datos desde un archivo CSV
  datos = pd.read_csv('datos.csv')

  # Realizar manipulaciones en los datos
  promocion = 0.15
  datos['columna_nueva'] = datos['columna_existente'] * (1-promocion)

  # Guardar los resultados en un nuevo archivo CSV
  datos.to_csv('datos_modificados.csv', index=False)

  print("Proceso de manipulación de datos completado.")

Este programa utiliza una biblioteca muy popular llamada pandas para manipular datos tabulares de manera eficiente. Puedes adaptar programas similares a tus necesidades específicas.

Automatización de Tareas Web con Selenium

Python también es útil para la automatización de tareas web, y la biblioteca Selenium es una herramienta valiosa para ello. Por ejemplo, si necesitas extraer información de una página web de forma regular, Selenium puede ayudarte a automatizar este proceso.

                  
  from selenium import webdriver
  from selenium.webdriver.common.keys import Keys
  import time
  
  # Iniciar el navegador web automatizado (por ejemplo, Chrome)
  navegador = webdriver.Chrome()
  
  try:
      # Abrir la página de Yahoo Finance
      navegador.get('https://finance.yahoo.com')
  
      # Encontrar el cuadro de búsqueda e ingresar el símbolo de una acción (por ejemplo, AAPL para Apple)
      cuadro_busqueda = navegador.find_element_by_name('p')
      cuadro_busqueda.send_keys('AAPL')  # Se puede cambiar 'AAPL' por el símbolo de la acción que sea
  
      # Presionar Enter para realizar la búsqueda
      cuadro_busqueda.send_keys(Keys.RETURN)
  
      # Esperar unos segundos para que se cargue la página con los resultados
      time.sleep(3)
  
      # Encontrar el elemento que contiene el precio de la acción
      precio_elemento = navegador.find_element_by_xpath('//span[@data-reactid="50"]')
      precio_accion = precio_elemento.text
  
      print(f"El precio de la acción es: {precio_accion}")
  
  finally:
      # Cerrar el navegador al finalizar
      navegador.quit()

Este ejemplo utiliza Selenium para abrir un navegador web automatizado, navegar a Yahoo Finance y extraer el precio de la acción de Apple. Usando pandas, un robot podría leer un CSV o archivo Excel, extraer una lista de acciones a buscar y completar la información en el mismo archivo. A esto se lo conoce como "Web Scraping", y aunque es una herramienta muy útil, a veces está sujeto a restricciones. Por eso en muchos casos se usa una API, un método más rápido y estable.
